# Healthrix System - Quick Reference Card

## For Employees

### Logging Activities Daily

1. **Open the App** (mobile or web)
2. **Click "Log Activity"**
3. **Fill in:**
   - Date (defaults to today)
   - Your name (auto-selected)
   - Task name (dropdown)
   - Count (how many times)
   - Patient ID (optional)
4. **Submit**

**When to Log:**
- Log activities as you complete them OR
- Log all activities at end of day
- Must log daily for accurate performance tracking

### Viewing Your Performance

1. Open app
2. Click "My Performance"
3. See your scores:
   - **Productivity Score** (max 90): Based on tasks completed
   - **Behavior Score** (max 10): Based on idle time & conduct
   - **Final Score** (max 100): Your total performance %

## For Supervisors

### Recording Daily Metrics

1. **Open the App**
2. **Click "Daily Metrics"**
3. **For each employee, record:**
   - Date
   - Employee name
   - Idle hours (e.g., 0.5, 1.0, 2.5)
   - Conduct flag:
     - **0** = No issues (normal)
     - **1** = Conduct issue flagged
   - Notes (if conduct flag = 1)
4. **Submit**

**When to Log:**
- End of each day
- Required for accurate performance calculation

### Viewing Team Performance

1. Open app
2. Click "Team Performance"
3. See leaderboard sorted by performance
4. Identify top performers and those needing support

## Performance Calculation Formula

```
Productivity Score (90% weight):
  Total Points = Sum of (Task Count √ó Task Base Score)
  Productivity % = (Total Points / 400) √ó 100
  Weighted = Productivity % √ó 0.90

Behavior Score (10% weight):
  Base = 100
  Minus: Idle Hours √ó 10 points
  Minus: Conduct Flag √ó 50 points
  Weighted = Base √ó 0.10

FINAL PERFORMANCE = Productivity + Behavior
```

## Common Questions

**Q: When are scores calculated?**
A: Automatically every night at 1:00 AM for the previous day

**Q: Can I edit my logged activities?**
A: Yes, you can edit your own entries (not others')

**Q: What if I forget to log activities?**
A: You can log activities for previous days (up to reasonable limit)

**Q: How can I improve my performance score?**
A:
1. Complete more high-value tasks (Authorization Created = 45 pts)
2. Meet or exceed daily target (400 points)
3. Minimize idle time
4. Maintain good conduct

**Q: Who can see my performance scores?**
A: You can see your own. Supervisors/managers can see everyone's.
